# Claude Design 작업 지시서 — 영상 히어로 + 일본 지도 내비게이션

## 배경
현재 사이트(`suntory-tatsujin-guide.html`)는 `<header class="masthead">`에 타이틀/서브타이틀/등급설명/범례가 바로 나옴(텍스트 밀도 높음). 그 위에 신규 섹션을 하나 추가한다.

## 요청 1: 영상 히어로 섹션
- masthead 바로 위, 새 `<section class="video-hero">` 추가.
- 배경 영상: `anime/japan5_bg_horizontal_16x9.mp4`(데스크톱), `anime/japan5_bg_vertical_9x16.mp4`(모바일, `≤620px`에서 스왑). `autoplay muted loop playsinline`.
- 텍스트 가독성 위해 영상 위 다크 그라디언트 오버레이 필수(영상 톤 밝음, 사이트는 다크테마 `--bg:#090806`).
- 오버레이 콘텐츠: 기존 h1 "위대한 AI님의 일본 맛집 가이드" + eyebrow "神泡達人店 · 산토리 인증 · 타베로그 검증 · 5개 도시 350곳" 중앙 배치. 그 아래 기존 masthead(서브타이틀/등급설명/범례)는 이 섹션과 별개로 그대로 유지 — 손대지 않음.
- 톤: 골드 액센트(`--accent:#e0a445`, `--accent-bright:#f5c069`) 포인트만, 나머지는 다크+영상.
- 폰트: `Zen Kaku Gothic New`, `Noto Sans KR` (기존 h1과 동일 스택 유지).
- 높이: 풀스크린 아님, 짧고 임팩트 있는 정도(뷰포트 40~60vh 권장, 확정 아님 — 시안에서 제안).

## 요청 2: 일본 지도 내비게이션 (참고 이미지 스타일)
- 참고: JNTO 스타일 인터랙티브 일본 지도(첨부 예시 이미지) — 전국 지역을 옅은 색으로 표시, 특정 지역만 강조색+빨간 라벨.
- 우리 사이트는 지역 전체가 아니라 **5개 도시만** 취급: 도쿄·교토·오사카·후쿠오카·삿포로. 지도에서 이 5곳만 활성(골드 라벨+마커), 나머지 지역은 회색/비활성(클릭 불가, 호버 없음) — 예시 이미지의 "빨간 라벨 4곳 + 나머지 회색" 구도를 우리 5곳 기준으로 재구성.
- 좌측 사진 캐러셀 카드(예시 이미지 왼쪽 부분)는 **선택 사항** — 필수 아님, 넣을 거면 도시 대표사진 슬라이드 정도로. 핵심은 우측 지도.
- **지도는 기존 도시 탭(`#cityChips`)을 대체하지 않음.** 탭은 그대로 유지, 지도는 보조 내비(비주얼) 역할.
  - 지도에서 도시 클릭 시: 결과 영역까지 스크롤 + 해당 도시로 필터 적용(기존 `#cityChips` 버튼 클릭과 동일한 동작 — 내부적으로 그 버튼을 트리거하거나 같은 로직 호출).
- 위치: 영상 히어로 섹션 안(또는 바로 아래) — 시안에서 자유롭게 제안, 확정된 건 masthead 앞/근처라는 것뿐.
- 다크테마 기준으로 디자인(라이트모드 토글도 있으니 두 톤 다 고려).

## 톤/제약 공통
- 기존 사이트 디자인 언어 유지: 다크 배경, 골드 액센트, 세리프성 고딕(Zen Kaku Gothic New), 배지pill 없는 플레인텍스트 스타일(1a/2a 시안 계승).
- 검색바 없음(의도적으로 제거된 상태, 부활 요청 아님).
- 이 작업은 프롬프트/시안 단계까지만 — 실제 HTML 반영은 별도 세션에서 진행.
